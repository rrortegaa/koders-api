# Koders API
#### Koders CRUD with authentication and authorization
## EndPoints
- GET     /       ->  get all koders
- GET     /:id    ->  get koder by id
- POST    /       ->  create koder
- PATCH   /:id    ->  update koder by id
- DELETE  /:id    ->  delete koder by id
- POST    /login  ->  login to access protected endpoints
